Configurá tu token de GitHub una sola vez y los cambios que guardes aparecerán automáticamente en todos los visitantes del sitio.

Solo la primera vez:
1. GitHub → tu foto → Settings
2. Menú izquierdo → Developer settings
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic) → nombre: allfit → tildá repo → Generate
5. Copiá el token (ghp_xxx...) y pegalo abajo
